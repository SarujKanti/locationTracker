Location Tracker is a modern Android application designed to track and share a user’s real time location efficiently and securely. The app integrates Google Maps to visually display the user’s current position and provides continuous background location tracking using a foreground service, ensuring reliability even when the app is minimized or closed.
The application follows best Android development practices, including proper permission handling, location services management, and a clean user interface optimized for different screen sizes.

**Key Features**
1. Live Location on Google Maps
2. Foreground Location Tracking
3. Runtime Permission Handling
4. GPS Enable Detection
5. Share Current Location
6. Persistent Notification
